DataMemory (MEMORY_TILE_MODULE) Register

DataMemory (MEMORY_TILE_MODULE) Register Description

Register NameDataMemory
Offset Address0x0000000000
Absolute Address

The base address for this register can be found by using the following formula:

    Register Base Address = 0x200_0000_0000 + (<colnum> * 32 + <rownum>) * 0x10_0000

Note: <colnum> and <rownum> are the column and row indices of the module in the matrix of AI Engine tiles.

Width256
TyperwNormal read/write
Reset Value0x00000000
DescriptionDataMemory of 8 banks, 256-bit wide, 512KiB
