Stream_Switch_Parity_Status (MEMORY_TILE_MODULE) Register

Stream_Switch_Parity_Status (MEMORY_TILE_MODULE) Register Description

Register NameStream_Switch_Parity_Status
Offset Address0x00000B0F10
Absolute Address

The base address for this register can be found by using the following formula:

    Register Base Address = 0x200_0000_0000 + (<colnum> * 32 + <rownum>) * 0x10_0000

Note: <colnum> and <rownum> are the column and row indices of the module in the matrix of AI Engine tiles.

Width32
TypewtcReadable, write a 1 to clear
Reset Value0x00000000
DescriptionStatus bits for Parity errors on stream switch ports (error on lower OR upper 32-bit word)
