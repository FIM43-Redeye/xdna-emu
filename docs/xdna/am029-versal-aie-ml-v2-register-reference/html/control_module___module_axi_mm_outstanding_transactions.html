Module_AXI_MM_Outstanding_Transactions (CONTROL_MODULE) Register

Module_AXI_MM_Outstanding_Transactions (CONTROL_MODULE) Register Description

Register NameModule_AXI_MM_Outstanding_Transactions
Offset Address0x00000C0024
Absolute Address

The base address for this register can be found by using the following formula:

    Register Base Address = 0x200_0000_0000 + (<colnum> * 32 + <rownum>) * 0x10_0000

Note: <colnum> and <rownum> are the column and row indices of the module in the matrix of AI Engine tiles.

Width32
TyperoRead-only
Reset Value0x00000000
DescriptionAXI-MM outstanding transactions monitors
