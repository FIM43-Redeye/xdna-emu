PL_Interface_Downsizer_Config (PL_MODULE) Register

PL_Interface_Downsizer_Config (PL_MODULE) Register Description

Register NamePL_Interface_Downsizer_Config
Offset Address0x0000030004
Absolute Address

The base address for this register can be found by using the following formula:

    Register Base Address = 0x200_0000_0000 + (<colnum> * 32 + <rownum>) * 0x10_0000

Note: <colnum> and <rownum> are the column and row indices of the module in the matrix of AI Engine tiles.

Width32
TyperwNormal read/write
Reset Value0x000006DB
DescriptionPL to AIE Interface Configuration (downsizing and/or upsizing)
