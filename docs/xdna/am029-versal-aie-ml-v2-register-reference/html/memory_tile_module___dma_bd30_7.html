DMA_BD30_7 (MEMORY_TILE_MODULE) Register Description

Register NameDMA_BD30_7
Offset Address0x00000A03DC
Absolute Address

The base address for this register can be found by using the following formula:

    Register Base Address = 0x200_0000_0000 + (<colnum> * 32 + <rownum>) * 0x10_0000

Note: <colnum> and <rownum> are the column and row indices of the module in the matrix of AI Engine tiles.
